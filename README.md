# 🚀 100 Days of Machine Learning

Welcome to my **100 Days of Machine Learning** challenge! 🎯

This repository documents my daily learning journey in Machine Learning, Data Science, and Python. Every day, I learn a new concept, implement it using Python, and upload the corresponding Jupyter Notebook.

## 📌 Objective

The goal of this challenge is to:

- Learn Machine Learning from scratch
- Build strong Data Science fundamentals
- Practice Python libraries used in ML
- Stay consistent for 100 days
- Share my progress publicly
